How to Install

To install CaseOh Bot, you need to download the source code and install it on your computer. Follow these steps:

  1. Clone the repository: git clone https://github.com/Janblocks1910/Case-Oh-Bot.git
  2. Navigate to the project directory: cd Case-Oh-Bot
  3. Install the dependencies: npm install qrcode-terminal path github:pedroslopez/whatsapp-web.js#webpack-exodus
  4. Make sure you have Node.js and npm installed on your computer.
  5. Use `wawebjs` to interact with WhatsApp Web. For more details, visit the Whatsapp-Web.js documentation.

How to Use

To use CaseOh Bot, you need to add the bot to your WhatsApp group and start chatting with him. Follow these steps:

  1. Run the bot: node index.js
  2. Scan the QR code with your WhatsApp to log in.
  3. Add the bot to your WhatsApp group.
  4. Send commands or interact with the bot to receive responses.
  5. For more details on using `wawebjs`, refer to the Whatsapp-Web.js documentation.
